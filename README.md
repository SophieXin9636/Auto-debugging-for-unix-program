# Auto-debug for UNIX homework

@ By Sophie Shin

## Why?
Just for fun! <br>
Easy debugging!

## Environment
* OS: Ubuntu 16.04 (or 18.04)
* Python2
* tcsh

## Install
* Python2 pip
```
sudo apt-get install python-pip
```
* pwntools
```shell
pip install pwntools
```
* tcsh
```
sudo apt-get install tcsh
```

## Run
input your file name (no extension)
```shell
cd auto_debug_for_unix
bash run.sh
```
